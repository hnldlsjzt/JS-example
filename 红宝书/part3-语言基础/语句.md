### 关系操作符(>、<、>=、<=)

转换过程：

1.两边都是数值，直接比较；

2.一边是数值，则将另一侧先尝试把字符串转为数值，再来比较；

3.都是字符串，则比较 ASCll 编码；

4.是对象，先调用 valueOf() ,没有改方法就调用 toString() 方法；

5.布尔值，则将其转换成数值来比较。

** NaN 和任何值相比较，都是 false **

```js
console.log('a' > 1, 'a' < 1, 'a' >= 1, 'a' <= 1) // 都是false
```

### 相等操作符(==)

null 和 undefined 相等，但不能和其他类型比较。
NaN 和任何值都不相等
